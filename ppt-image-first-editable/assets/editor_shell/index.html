Phase C — 文字编辑器

在背景图的预留区域内调整文字内容、位置与样式

页面

提示：单击文字框 → 拖动；双击 → 编辑文字；右下角点拖 → 改尺寸；Esc → 退出编辑；Delete → 删除选中

属性

点击画布上的文字框可编辑其属性。

导出 / 继续生成

你有两个出口：
1. 不再编辑，直接用当前系统给的 deck.json 生成 PPTX
2. 继续编辑后，把整段内容（带标记）粘回对话框
⚠️ 重要：把下面整段（带 ===DECK JSON=== 标记的全部内容）发回给 AI， AI 看到标记会知道要保存 + 渲染 PPTX。
不要只复制中间的 JSON 部分；不要另起一句话发出去； 直接整段贴到 AI 对话框即可。

使用说明

  • 左侧切换页面，中间是当前页的画布，右侧编辑选中文字框的属性
  • 单击文字框 → 选中；选中状态拖动 = 移动；右下角点拖 = 缩放
  • 双击文字框 → 进入文字编辑（contentEditable），Esc 退出
  • 选中后按 Delete / Backspace（非编辑态）删除该文字框
  • 顶栏 + 添加文字框 在画布中央插入一个新的
  • 所有坐标都是 0-1 的 fraction（左上为原点），跟 PPTX 渲染器同语义
  • 字体只在 PingFang/Microsoft YaHei/Hiragino/Arial/Helvetica 等系统安全字体里选，避免 PPTX 跨平台跑偏
  • 改完后可以直接生成，也可以把整段 deck 贴回对话框再生成
